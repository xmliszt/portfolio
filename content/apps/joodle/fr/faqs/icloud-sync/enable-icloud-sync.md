---
id: enable-icloud-sync
title: Comment activer la synchronisation iCloud ?
order: 0
---

**La synchronisation iCloud est gratuite pour tous les utilisateurs !** Pour l’activer :

1. Assurez-vous d’être connecté à iCloud sur votre appareil.
2. Activez iCloud pour Joodle dans **Réglages → [Votre nom] → iCloud → Enregistré dans iCloud → Joodle**.
3. Activez la synchronisation dans l’app Joodle via **Réglages → Général → Synchronisation iCloud**.

**Important :** l’option iCloud système et l’option dans l’app doivent toutes les deux être activées pour que la synchro fonctionne.

Après activation, il peut être nécessaire de redémarrer l’app pour que les changements prennent effet.
