title: 프로모 코드를 어떻게 사용하나요?
order: 3
---

**프로모 코드를 사용하려면 다음 단계를 따르세요:**

1. 기기에서 Joodle 앱을 엽니다.
2. "설정" 페이지로 이동합니다.
3. 맨 위에 "Joodle Pro"가 있는 구독 섹션이 표시됩니다. 그 아래의 "프로모 코드 사용" 옵션을 탭합니다.
4. 제공된 필드에 프로모 코드를 입력합니다.
5. "계속"을 탭하여 코드를 적용하고 지시를 따릅니다.

**참고**: 프로모 코드는 한 번만 사용 가능한 코드입니다. 일부 코드는 무료 체험과 같은 기존 소개 혜택에 추가 효과가 있어 구독 기간이 연장될 수 있습니다. 프로모 코드를 적용하는 동안 구독을 취소하면 Apple의 정책에 따라 현재 Joodle Pro 구독이 즉시 손실될 수 있습니다.
