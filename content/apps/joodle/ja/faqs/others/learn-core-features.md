Joodleには、主要機能を自分のペースで習得できるインタラクティブなチュートリアルが用意されています。

**インタラクティブチュートリアルの開き方:**

1. **Settings > Need Help? > Learn Core Features** へ移動します
2. 利用できるチュートリアルの一覧を表示します
3. 学びたいチュートリアルをタップして開始します

各チュートリアルでは、Joodleの特定のパートを実際に手を動かしながら学べます。

**オンボーディングのやり直し:**

Learn Core Features画面の下部から、最初に見逃した方や、もう一度確認したい方に向けてオンボーディング体験をやり直すことができます。

⚠️ **重要な注意:** オンボーディングをやり直すと、本日のJoodleエントリーが上書きされ、各種設定の選択もやり直しになります。問題ないことを確認してから進めてください。
